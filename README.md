# RecoilStrafeTrainer

RecoilStrafeTrainer 是一个基于 Electron、React 和 Vite 构建的桌面训练工具，用于练习压枪节奏与横移方向提示。

它提供原生全局键鼠监听、悬浮小窗、置顶与点击穿透、武器参数编辑、时间轴可视化，以及亮色 / 暗色主题切换，适合在桌面环境中进行快速训练和调试。

## 功能特性

- 原生全局监听
  支持在 Electron 桌面环境下监听全局键盘与鼠标输入，不依赖网页焦点。

- 监听式触发逻辑
  点击开始后进入监听状态，按下你设置的触发键时播放一轮后坐力提示；松开后停止，再次按下会重新开始一轮。

- 自定义触发键
  支持把监听键设置为常见键盘按键，或鼠标左键 / 中键 / 右键。

- 小窗模式
  可将主页面切换为悬浮小窗，便于覆盖在其他窗口上进行训练。

- 置顶与点击穿透
  小窗支持始终置顶，并可开启点击穿透，让鼠标操作直接落到后方窗口。

- 小窗快捷键控制
  默认提供以下快捷键：
  - `F8`：退出小窗
  - `F7`：切换置顶
  - `F6`：切换点击穿透

- 武器参数编辑
  可直接在界面中编辑预设武器的名称、RPM、弹匣容量与拐点配置。

- 横移节奏时间轴
  通过可视化时间轴展示不同阶段的横移方向，便于快速核对整轮节奏。

- 配置导入 / 导出
  支持将武器配置导出为 JSON，也可以导入已有配置文件进行覆盖或追加。

- Light / Dark 主题
  内置亮色与暗色主题，方便适配不同桌面环境。

## 使用说明

### 1. 选择武器

启动应用后，左侧会显示武器列表。

- 点击任意武器可切换当前训练对象
- 右侧面板会同步显示该武器的基础参数与横移节奏

### 2. 编辑武器参数

在主面板中点击“编辑”后，可以修改：

- 武器名称
- `RPM`
- 弹匣容量
- 拐点列表

拐点的含义是：
从第几发开始，提示切换为某个方向。

例如：

```text
第 1 发开始向右
第 10 发开始向左
第 18 发开始向右
```

保存后会立即应用到当前配置。

### 3. 设置监听键

在控制区可以重新设置监听键。

设置方式：

- 点击“重新设置”
- 按下你想要作为触发键的键盘按键，或鼠标按键
- 按 `Esc` 可取消这次设置

### 4. 开始监听

点击“开始”后，应用不会立刻无限播放，而是进入监听状态。

当前逻辑为：

1. 点击开始
2. 应用进入监听状态
3. 按下触发键后，开始播放一轮后坐力 / 方向提示
4. 松开触发键后停止
5. 再次按下触发键，会重新开始新一轮

这套逻辑更适合和实际训练动作同步。

### 5. 调整训练参数

控制区支持调整以下内容：

- 触发延迟
- 音量
- 小窗缩放
- 小窗透明度

### 6. 查看横移节奏

底部“横移节奏”区域会以颜色条展示当前整轮的方向变化：

- 蓝色：左
- 红色：右
- 紫色：下

它可以帮助你快速判断整个弹匣的节奏结构是否合理。

### 7. 进入小窗模式

点击“小窗”后，主页面会切换到悬浮小窗模式。

小窗模式适合放在游戏或其他窗口上方，用于实时方向提示。

### 8. 小窗快捷键

默认快捷键如下：

- `F8`：退出小窗
- `F7`：切换始终置顶
- `F6`：切换点击穿透

点击穿透的含义是：
小窗仍然显示在屏幕上，但鼠标点击会直接穿过它，作用到后面的窗口。

## 配置管理

### 导出配置

点击控制区的导出按钮，可以将当前所有武器配置导出为 JSON 文件。

适合用于：

- 备份配置
- 在不同电脑之间迁移
- 分享训练方案

### 导入配置

点击导入按钮后，可选择本地 JSON 文件进行导入。

导入规则：

- 如果 `id` 已存在，则覆盖已有武器配置
- 如果 `id` 不存在，则追加为新武器

## 预设武器在哪里修改

如果你想直接改源码中的默认预设，可以查看：

- `src/data.ts`

这里定义了默认武器列表、基础参数和拐点配置。

如果你已经在应用里改过配置，应用会优先读取本地保存的数据，而不是直接回退到源码默认值。

## 常见问题

### 为什么我点击开始后没有立刻播放？

这是当前项目的设计逻辑。
点击开始只是进入监听状态，只有按下你设置的监听键时，才会触发一轮提示。

### 为什么松开按键后会停止？

因为当前训练逻辑是按住触发、松开停止、再次按下重新开始，避免原来那种无限循环播放。

### 小窗为什么有时候点不到？

如果你开启了点击穿透，小窗就不会接收正常鼠标点击。
这时请使用快捷键切换状态，例如按 `F6` 关闭点击穿透。

### 可以直接新增自己的枪械方案吗？

可以。
你可以通过导入 JSON 新增方案，也可以直接修改源码中的默认预设。
